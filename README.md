# Estudio Violeta

Landing page estática para Estudio Violeta, un emprendimiento de papelería personalizada, souvenirs e impresión en San Luis.

El sitio presenta sus trabajos, servicios, formas de pedido, ubicaciones y condiciones comerciales, con WhatsApp como canal principal de contacto y Compre San Luis como catálogo secundario.

## Sitio publicado

[estudio-violeta.netlify.app](https://estudio-violeta.netlify.app/)

## Tecnologías

- HTML semántico
- CSS responsive
- JavaScript vanilla
- Imágenes optimizadas en WebP

## Estructura

- `index.html`: contenido, SEO y datos estructurados.
- `styles.css`: diseño visual y adaptación responsive.
- `script.js`: menú mobile, animaciones de aparición y enlaces dinámicos de WhatsApp.
- `assets/`: logo, favicon y fotografías de trabajos.
- `robots.txt` y `sitemap.xml`: configuración básica para buscadores.

No requiere instalación de dependencias ni proceso de build.
